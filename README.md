# 重庆高校 Linux 上网客户端 openkeeper

## Introduction

- An open source implementation of Netkeeper for Linux.

## Build & install dependencies

```
Ubuntu: pppoe curl pppd
Debian: pppoe curl pppd
openSUSE: rp-pppoe curl pppd
```

## Installation

```bash
$ cd openkeeper-cli
$ sudo make install
```

## Usage

- Configuration

```bash
$ sudo ok-config
```

- Connect

```bash
$ sudo ok
```

- Disconnect

```bash
$ sudo ok-stop
```
